# Myntra Clone Website

A responsive e-commerce website inspired by **Myntra**, built using **HTML, CSS, and JavaScript**.  
This project replicates core features of an online fashion store, showcasing frontend development skills and UI/UX design.

---

## 🚀 Features
- Responsive design for desktop and mobile devices
- Product listing with categories and filters
- Interactive shopping cart (add/remove items, real-time price updates)
- Dynamic search functionality
- Wishlist and product detail pages (optional if implemented)
- Clean and structured code using HTML5, CSS3, and JavaScript (ES6)

---

## 🛠️ Technologies Used
- **HTML5** for semantic structure  
- **CSS3** (Flexbox & Grid) for styling and layout  
- **JavaScript (ES6)** for dynamic interactions and DOM manipulation  
- **Local Storage** for cart persistence (if implemented)
